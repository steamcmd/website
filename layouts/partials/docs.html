Docs

Versioning

The API is versioned for compatibility reasons, but only /v1/ is currently available.

Authentication

No authentication or verification is required to use the API. Currently no (tight) rate limiting or throttling is implemented. However, caching will be added in the future to avoid unnecessary requests to Valve servers.
